Worldwide Weather

This is the last assignment in the Thinkful Front End Developer course. I really enjoyed this one. It was actually quite easy to put together with some features that I thought were pretty cool. I used three APIs to make this app work:

  • A plugin for Google Places with auto-complete search
  • The API from Weather Underground to get current and forecast weather from around the world
  • And the Flickr API to get a background image of the place that the user searched for

Some improvements that I could make with this are:

  • Lookup the user's current location and display weather for where they are, so that the page doesn't start blank
  • Make the user experience for mobile better - perhaps using Bootstrap

Stack Overflow API

This Thinkful project started from an existing GitHub repository. The focus was purely on the API. This app uses the Stack Exchange API to get data from the Stack Overflow site.

Two different end points were used, one to get unanswered questions regarding a particular keyword. The other to get the top answerers (is that even a real word?) for a particular keyword.

Quiz App

In this Thinkful project, I learned about how to implement Javascript objects in a practical example.

This quiz has five static questions. Each question is implemented as a Javascript object. When the user clicks the Next Question button, the next question object is used to update the same page. Through the use of Javascript objects, the quiz could be scaled with many more questions and then have the five questions picked at random.

Hot or Cold

This project started with the front end already done and we were to implement the Javascript.

This app generates a random number between 1 and 100, then takes the user's guess and gives feedback on how close the guess was. The guesses are listed at the bottom and the number of guesses are counted. The app also checks if the guess is actually a number and if it is within the correct range.
