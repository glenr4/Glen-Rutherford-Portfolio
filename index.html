Worldwide Weather

This is the last assignment in the Thinkful Front End Developer course. I really enjoyed this one as it was actually quite easy to put together with some features that I thought were pretty cool. I used three APIs to make this app work:

  • A plugin for Google places with auto-complete place search
  • The API from weather underground to get current and forecast weather from around the world
  • And the Flickr API to get a background image of the place that the user searched for

Some improvements that I could make with this are:

  • Lookup the users current location and display weather for where they are so that the page doesn't start blank
  • Make the user experience for mobile better

Karma Replica

The object with this project in the Thinkful Front End Developer course was to get the look, layout and fonts virtually the same as the example website from Karma Mobility.

See if you can pick the original.

We had to run this on a webserver to get the font to load, so I fired up my WAMP server on my computer. And of course it didn't work! After a google search for the error that came up, there was a conflict on port 80. So then I had to track down the culprit with netstat. I found the process ID that was listening on port 80 and found out it was Skype! So I shutdown Skype and restarted the services on WAMP and it went green - good to go.

Thinkful Front End Web Development Course

I realised a few things about my web development experiences:

  1. I had rekindled my love of programming with something that was visually and functionally rewarding
  2. I was frustrated with my level of knowledge not being at the point where I could create what I wanted to create
  3. My learning path was unplanned and I had no idea what the industry norms were for doing common tasks

So I started looking around for more online courses. I really wanted to do a bootcamp but at 12+ weeks fulltime, while looking after my family, it just wasn't an option.

After looking through seek.com to see what skills companies are looking for in web development, it really comes down to three core skills:

  1. html
  2. css
  3. javascript

Pretty much everything else builds on top of this. Then there is the server side stuff like php, ruby etc. but I thought, well I have to get the fundamentals down first, then I can branch out further.

When I found the Thinkful Front End Developer course it ticked all of the boxes of what employers look for:

  • html, css, javascript
  • Using github to backup and handle version control
  • Create a portfolio of projects to demonstrate coding ability and making stuff work

The price wasn't excessive and it also came with a mentor who I can talk to one on one once a week. So I thought I haven't found anything better than that, sign me up!

The early days

Well after some creating some websites firstly with Joomla, then Wordpress I got fed up with the restrictions that they or the theme that was installed posed. There was always something that I wanted different to what they allowed me to do. As someone who solves technical problems for a living, I got annoyed with this and thought there has to be a better way.

I eventually came across bootstrap what a revelation! How easy was it to use, responsive and had lots of animated effects built in!

After creating a few pages I wanted to change something in the navigation bar. Then I realised what a pain it is having to maintain the same pieces of code on each page. I did some googling but didn't find anything on how other people had solved this problem. So I thought, well the only actual related language that I know that could help here is php. So I created a navigation file in php and used the include function in my html, then I only had to maintain it in one place.

When I was experimenting with Joomla and Wordpress, I initially set them up at home on a WAMP server. This was a relatively easy process but there are a few things that needed to be done to make it work:

  • Setup virtual servers so that the links have the correct path when deployed to the web
  • Editing hosts file to map my websites to localhost
  • Security to limit connections from my LAN only
